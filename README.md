# 🛒 쇼핑 리스트 앱

HTML, CSS, JavaScript만으로 만든 간단한 쇼핑 리스트 웹 앱입니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 아이템 완료 체크
- 아이템 개별 삭제
- 완료된 항목 일괄 삭제
- 완료 진행률 표시 (N / 전체 완료)
- 새로고침 후에도 데이터 유지 (localStorage 사용)

## 사용 방법

별도 설치 없이 `shopping-list.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage (데이터 저장)
